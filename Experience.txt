This file is for me to seperate out some experiences in using claude. There are different kinds of uses. I will try jot them down when I encounter them, and if there's interest I'll try to work them out at some point. 

- Boilerplate code writing (example derive CommRingHom from \neg \wedge), good
- looking up definitions (where was the function stating that two props are equal if they are equivalent?) quickly. Not sure if it's quicker if you know the code better, but useful for cubical library for me. 
- Formalizing the paper, really depends on how detailed the original proof was.
- Filling in holes, not sure yet, it needed some assistence for NFinCofin presentation. But also that was brute force work and would have taken ages for me. 
- If I have myself done some instances for a general proof (but was to stupid to realize it / didn't see the forest trough the trees) then it could do the general thing if I prompt it to. Similarly, it was good with doing plumbing.
